# Conditions

If condition is met, schema is merged with schema inside then. Minimum length (minLength) and equality (const) conditions are supported. Also oneOf (condition is met if only one if is met) and allOf (condition is met if all ifs are met) combinations can be used.

# Default Values

Sometimes, usually when initializing a new data instance, you would want to use some default values. Full data model object with default values is emitted after initialization through a @init-default event.

<JsonSchema :schema="schema" v-model="dataModel" @init-default="dataModel = $event"/>
